# Lean Canvas: NutriBot

> **Дата создания:** 2026-01-30
> **Версия:** 1.0
> **Основано на:** Brief v1.0

---

## 1. Problem (Проблема)

### Топ-3 проблемы:

1. **Рутина сбора анамнеза:** Нутрициолог тратит 40-60 минут на первичный опрос каждого клиента — это время, которое можно потратить на экспертизу.

2. **Ручная обработка анализов:** Нужно вручную сопоставлять показатели с нормами, искать паттерны, фиксировать отклонения — монотонная работа с высоким риском ошибки.

3. **Ограничение масштаба практики:** Количество клиентов ограничено временем специалиста. Больше клиентов = меньше качества или выгорание.

### Existing Alternatives (как решают сейчас):

- **Google Forms / Typeform:** Сбор данных есть, но нет анализа, нет интерактивности
- **Excel / Notion:** Ручное ведение записей, разрозненные данные
- **Просто в переписке:** Telegram/WhatsApp — данные теряются, нет структуры
- **Ничего:** Опрос устно на консультации — неэффективно

## 2. Customer Segments (Сегменты клиентов)

### Целевой сегмент:

- **Кто:** Практикующие нутрициологи и диетологи
- **Характеристики:**
  - 10-50 активных клиентов в месяц
  - Работают индивидуально или в небольшой практике
  - Используют Telegram для коммуникации с клиентами
  - Готовы инвестировать в инструменты для экономии времени
  - Возраст: 28-45 лет, преимущественно женщины (80%)

### Early Adopters (кто купит первым):

- **Технически продвинутые нутрициологи:** Уже используют онлайн-инструменты, ведут Instagram/Telegram-каналы
- **Нутрициологи с растущей практикой:** Ощущают нехватку времени, ищут способы масштабироваться
- **Нутрициологи-энтузиасты AI:** Интересуются ChatGPT, экспериментируют с промптами

---

## 3. Unique Value Proposition (UVP)

### Главное обещание:

> **AI-ассистент, который собирает анамнез и анализирует данные клиента за вас — вы фокусируетесь на экспертизе, а не на рутине.**

### High-Level Concept:

> "**Notion AI для нутрициологов**" — умный помощник, который превращает сырые данные клиента в готовый отчёт с рекомендациями.

---

## 4. Solution (Решение)

### Топ-3 функции:

1. **AI-опросник в Telegram:** Бот задаёт вопросы о питании, здоровье, целях — понимает свободный текст, адаптирует диалог.

2. **Умная обработка анализов:** Клиент загружает фото/PDF → система распознаёт показатели, сопоставляет с нормами, выделяет отклонения.

3. **Автоматический черновик отчёта:** LLM анализирует все данные и генерирует структурированный отчёт с интерпретацией и рекомендациями.

### MVP Scope:

- [x] Telegram-бот для клиентов (сбор анамнеза)
- [x] OCR + LLM для обработки анализов
- [x] Веб-панель для нутрициолога (карточки клиентов, отчёты)
- [x] Генерация PDF-отчёта
- [ ] ~~Планы питания~~ (v2.0)
- [ ] ~~Интеграция с лабораториями~~ (v2.0)

## 8. Key Metrics (Ключевые метрики)

### North Star Metric:

> **Количество обработанных клиентов в месяц** — показывает реальное использование и ценность для нутрициологов.

### Воронка и метрики:

| Этап | Метрика | Target (MVP) |
|------|---------|--------------|
| **Acquisition** | Регистрации нутрициологов | 20/мес |
| **Activation** | Создали первого клиента | 70% |
| **Retention** | Активны через месяц | 60% |
| **Revenue** | Конверсия в платную подписку | 30% |
| **Referral** | Привели коллегу | 20% |

### Дополнительные метрики:

- **Activation:** Время до первого обработанного клиента
- **Engagement:** Клиентов на нутрициолога в месяц
- **NPS:** Net Promoter Score > 40

## 9. Unfair Advantage (Некопируемое преимущество)

### Что нельзя скопировать:

1. **Экспертиза нутрициолога-основателя:** Глубокое понимание workflow и болей (инсайдер в отрасли)

2. **Накопленная база промптов и правил:** Итерации с реальными клиентами создают know-how, который сложно воспроизвести

3. **Сетевой эффект:** Чем больше нутрициологов используют → больше данных → лучше модель

### Если пока нет — план создания:

- **Месяц 1-3:** Глубокая работа с 5 пилотными нутрициологами → экспертиза
- **Месяц 3-6:** Накопление базы кейсов и оптимизация промптов
- **Месяц 6-12:** Партнёрства со школами → дистрибуция
