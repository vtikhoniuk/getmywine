# Brief: GetMyWine

> **Версия:** 1.0
> **Дата:** 2026-02-01
> **Статус:** Черновик

---

## Что нужно сделать?

**Описание задачи:**
Создать веб-сервис с GetMyWine, который помогает людям, не разбирающимся в вине, находить подходящие напитки через диалог. Сервис изучает вкусовые предпочтения пользователя и даёт персонализированные рекомендации из собственного каталога.

**Job Story:**
Когда я хочу купить вино, но не разбираюсь в нём и боюсь ошибиться, я хочу получить персональную рекомендацию от AI, который знает мой вкус, чтобы наслаждаться вином без стресса выбора.

---

## Полезное действие (Зачем?)

### Базовая проблема пользователя:

**Для потребителя:**
- Плохо разбирается в вине — не знает сорта, регионы, стили
- Не понимает собственный вкус — не может сформулировать, что нравится
- Теряется в ассортименте — сотни бутылок, непонятные этикетки
- Боится ошибиться — страх потратить деньги на вино, которое не понравится

### Ценность для бизнеса:

- Ниша «обучение через практику + персональный сомелье» слабо занята
- Потенциал для монетизации через партнёрство с винотеками, подписки, комиссии с продаж
- Подготовка к интеграции с UCP для автоматических покупок в будущем
- Высокое удержание — пользователь «растёт» вместе с сервисом

## Приоритет (RICE)

| Параметр | Значение | Обоснование |
|----------|----------|-------------|
| **Reach** | Средний | 30-45 лет, новички в вине, Россия/СНГ |
| **Impact** | 3 (High) | Решает реальную боль — страх и неуверенность при выборе вина |
| **Confidence** | 50% | Гипотеза не проверена на пользователях |
| **Effort** | 1 (Low) | MVP с минимальным функционалом |

**RICE Score:** Высокий — хороший кандидат для быстрого запуска и валидации

## Критерии приёмки

### MVP:
- [ ] Пользователь может зарегистрироваться и войти
- [ ] Пользователь может написать вопрос в чат и получить ответ от AI
- [ ] AI рекомендует вина из каталога (не выдумывает)
- [ ] Отображаются 1-3 персонализированные подсказки
- [ ] Подсказки меняются в зависимости от истории диалога
- [ ] В каталоге минимум 50 вин с базовой информацией

### Quality Gates:
- [ ] AI не галлюцинирует — рекомендует только вина из каталога
- [ ] Рекомендации релевантны запросу пользователя
- [ ] Профиль сохраняется между сессиями

---

## Out of Scope

### Не входит в MVP:
- Автоматическая покупка / оплата
- Интеграция с UCP
- Сканирование этикеток
- Обучающие курсы / геймификация
- Мобильное приложение (только веб)
- Мультиязычность
- Социальные функции (отзывы, друзья, подписки)
- Интеграция с внешними каталогами (Vivino, винотеки)

### Сознательные ограничения:
- Сервис не продаёт алкоголь напрямую
- Сервис не доставляет вино
- Рекомендации носят информационный характер

## Оценка рисков

### Продуктовые риски:
| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| AI галлюцинирует (выдумывает вина) | Средняя | Высокое | Строгая привязка к каталогу, RAG-подход |
| Персонализация работает плохо на малых данных | Средняя | Среднее | Продуманный онбординг, сбор предпочтений через диалог |
| Пользователи не понимают ценность | Низкая | Высокое | Чёткое позиционирование, быстрый first value |

### Бизнес-риски:
| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| UCP недоступен в России | Высокая | Среднее | Начать с рекомендаций, монетизация через партнёрства |
| Конкуренция с Vivino | Средняя | Среднее | Дифференциация: обучение + персонализация через диалог |
| Низкий retention | Средняя | Высокое | Путь развития вкуса, накопление профиля |

### Общий вывод:
Риски управляемые. MVP фокусируется на валидации ключевой гипотезы: «люди хотят GetMyWine, который учит их вкус через диалог». Минимальный scope позволяет быстро проверить идею.

## Контекст рынка

### UCP (Universal Commerce Protocol)
- Запущен Google в январе 2026
- Позволяет AI-агентам автономно совершать покупки
- Phase 1 — только США, международная экспансия — конец 2026
- Партнёры: Shopify, Stripe, Visa, Mastercard, Walmart, Target

### Конкуренты
| Игрок | Сильные стороны | Слабые стороны |
|-------|----------------|----------------|
| Vivino | 200M+ оценок, большая база | Не разговорный, много кликов |
| Preferabli | ML + эксперты-сомелье | B2B фокус |
| Winelikes | Геймификация обучения | Нет покупки |
| Sommelier.bot | Чатбот для e-commerce | Работает на магазин, не на пользователя |

### Наша дифференциация
Комбинация «Гид для новичков» + «Персональный сомелье»:
1. **Вход:** человек не знает, что ему нравится
2. **Путь:** AI ведёт через серию открытий
3. **Результат:** понятный вкусовой профиль
4. **Удержание:** персональный сомелье для регулярных покупок
