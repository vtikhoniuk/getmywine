## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Header | Логотип (→ Dashboard), поиск, уведомления, профиль |
| 2 | Sidebar | Навигация, активный пункт подсвечен |
| 3 | Карточки "Требуют внимания" | Клик → карточка клиента. Показываем до 3 штук |
| 4 | Статистика | Обновляется в реальном времени |
| 5 | Последние действия | Лог активности клиентов |
| 6 | Реферальная ссылка | Кнопка копирует в буфер, показывает toast "Скопировано" |

## Переходы

- Клик на карточку клиента → Карточка клиента
- Клик на "Клиенты" в sidebar → Список клиентов
- Клик на "Настройки" → Страница настроек
- Клик на профиль → Dropdown (Профиль, Выход)
