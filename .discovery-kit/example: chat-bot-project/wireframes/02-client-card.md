## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Breadcrumb | Клик на "Клиенты" → список клиентов |
| 2 | Avatar + контакты | Telegram username кликабелен (открывает чат) |
| 3 | Red flags | Показываются только если есть, красный фон |
| 4 | Tabs | Переключение без перезагрузки страницы |
| 5 | Карточка анализа | Клик → развёрнутый список показателей |
| 6 | Кнопка "Создать отчёт" | Запускает генерацию, показывает прогресс |

## Переходы

- Клик на tab → переключение контента
- Клик на анализ → детальный просмотр показателей (modal)
- "Создать отчёт" → страница редактора отчёта
- "Открыть" отчёт → страница редактора отчёта
