# Wireframe: Редактор отчёта

> **Платформа:** Desktop (1200px+)
> **Контекст:** Нутрициолог редактирует сгенерированный отчёт

## Аннотации

| # | Элемент | Поведение |
|---|---------|-----------|
| 1 | Tabs (Редактирование/Превью) | Мгновенное переключение |
| 2 | Кнопка [✎] | Открывает modal редактирования секции |
| 3 | Текстовое поле | Markdown-редактор с форматированием |
| 4 | "+ Добавить секцию" | Добавляет пустую секцию с заголовком |
| 5 | Превью PDF | Интерактивный просмотр страниц |
| 6 | "Сохранить черновик" | Сохраняет без генерации PDF |
| 7 | "Генерировать PDF" | Создаёт PDF (async, показывает прогресс) |
| 8 | "Отправить клиенту" | Отправляет PDF в Telegram, меняет статус |

## Переходы

- Breadcrumb → назад к карточке клиента
- "Сохранить черновик" → toast "Сохранено"
- "Генерировать PDF" → переключение на tab "Превью PDF"
- "Отправить клиенту" → confirmation modal → отправка → success message
