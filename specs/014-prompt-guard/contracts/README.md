# API Contracts: 014-prompt-guard

Эта фича **не добавляет новых API-эндпоинтов** и **не изменяет существующие контракты**.

Все изменения происходят внутри бэкенда:
- Системный промпт (инструкции к LLM)
- Парсер ответов LLM (in-memory)
- Логирование (Python logging)

Формат ответов для клиентов (web, Telegram) не меняется — маркер `[GUARD]` стрипается до отправки.
