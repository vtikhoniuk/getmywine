# GetMyWine

Телеграм-бот GetMyWine, который помогает людям, не разбирающимся в вине, находить подходящие напитки через диалог.

## Проблема

- Люди не разбираются в вине — сорта, регионы, стили
- Не понимают собственный вкус — не могут сформулировать, что нравится
- Теряются в ассортименте — сотни бутылок, непонятные этикетки
- Боятся ошибиться — страх потратить деньги на вино, которое не понравится

## Решение

GetMyWine:
- Ведёт диалог и изучает вкусовые предпочтения
- Даёт персонализированные рекомендации из каталога
- Помогает «расти» в понимании вина

## MVP

- Регистрация и авторизация
- Чат с GetMyWine
- Рекомендации из каталога (без галлюцинаций)
- Персонализированные подсказки на основе истории и предпочтений пользователя
- Каталог 50+ вин

## Стек

- Телеграм-бот
- LLM API для диалога
- RAG для привязки к каталогу

## Ограничения

- Сервис не продаёт и не доставляет алкоголь
- Только рекомендации
- 18+
